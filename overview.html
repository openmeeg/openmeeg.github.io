Overview¶

What is OpenMEEG?¶

  • The OpenMEEG software is developed within the Athena project-team at INRIA Sophia-Antipolis.
  • It was initiated in 2006 by the Odyssee Project Team (INRIA/ENPC/ENS Ulm).
  • OpenMEEG solves forward problems related to Magneto- and Electro-encephalography (MEG and EEG).
  • The forward problem uses the symmetric Boundary Element Method (symmetric BEM), providing excellent accuracy (see Publications).

Using OpenMEEG¶

  • From Matlab using Brainstorm or Fieldtrip

  • For general lead fields computation (EEG, MEG, EIT, Internal potential):

    • From Python
    • From a Bash script on Unix systems (Linux/Mac).
    • From a BAT file on Windows.
